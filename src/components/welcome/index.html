Welcome to WebRx Examples


This application provides a browsable collection of examples implemented using the WebRx Framework.

The application also acts as a complex real-world example of the Framework's Client-Side Routing capabilities utilizing multiple nested and individually animated views. The Routing-State Monitor Panel at the bottom of the page visualizes the router's internal state at all times.

If you have an example of which you think that it might be a good fit for this collection, feel free to submit a Pull-Request.
