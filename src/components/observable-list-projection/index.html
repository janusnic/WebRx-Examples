Example: Observable-List-Projections

This example utilizes WebRx's very useful observable-list-projections. A projected Observable List is a read-only view of of a source observable list that is automatically kept in sync with its source. What makes these projection so useful is that list-projections can apply filtering, re-ordering and mapping operations on the fly.

The planetsToShow projection used in this example filters the planets list to only include planets of a user selected category.
